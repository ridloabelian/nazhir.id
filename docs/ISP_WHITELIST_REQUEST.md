# Permohonan Normalisasi / Whitelist Domain nazhir.id

Kepada Tim Internet Baik / Trust & Safety ISP,

Mohon dilakukan normalisasi akses domain berikut:

- https://nazhir.id
- https://www.nazhir.id

Domain ini adalah situs resmi **Dashboard Wakaf Nasional Asosiasi Nazhir Indonesia (ANI)** untuk mendukung digitalisasi tata kelola wakaf, pelaporan aset wakaf, akuntansi PSAK 412, dan laporan dampak sosial lembaga Nazhir.

Saat ini beberapa pengguna melaporkan akses melalui jaringan seluler/ISP tertentu ter-redirect ke halaman filtering ISP, sehingga HTTPS gagal dengan pesan sertifikat seperti:

- `NET::ERR_CERT_COMMON_NAME_INVALID`
- sertifikat yang tampil bukan `nazhir.id`, melainkan domain filtering ISP.

Mohon domain ini dikategorikan sebagai:

- Education
- Organization / Nonprofit
- Religious / Islamic social finance
- Productivity / SaaS

Bukti teknis dari jaringan global:

```txt
DNS: nazhir.id dan www.nazhir.id mengarah ke Cloudflare
TLS issuer: Google Trust Services
Subject/SAN: nazhir.id, www.nazhir.id
HTTP status: 200
```
